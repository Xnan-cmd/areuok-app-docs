# 还好么are u ok - App Store 文档

> 一个温暖的签到应用，每天告诉家人你还好

---

## 应用介绍

**还好么are u ok** 是一款温暖的每日签到应用。每天点一下光球，告诉爱你的人你还在。如果超过 48 小时没有签到，我们会通知你信任的联系人。

### 核心特点

- 简单：一键签到，没有复杂功能
- 温暖：根据不同时段显示不同的温暖问候
- 安心：48小时未签到自动通知紧急联系人
- 隐私：不收集位置、通讯录等敏感信息

---

## 功能截图

### 主界面

会呼吸的温暖光球，点击即可签到

### 设置页面

设置紧急联系人和每日提醒时间

### 签到后

收到根据时段变化的温暖文案
